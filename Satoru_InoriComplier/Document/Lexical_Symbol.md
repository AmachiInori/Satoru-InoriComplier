# Satoru词法分析器

## 运算符

Satoru运算符用于组成表达式。优先级数值越小者越晚被结合。

## 运算符定义

### 0级运算符

- `,` 用于分隔表达式，左结合。被分隔的表达式列之返回值为首个表达式的值。

    ```
    a = 3, a + 3, sin(a); #a == 3#
    ```

### 1级运算符

1级运算符均与赋值相关，总共6个: `= += -= *= /= %=`，其中: 

- `=`用于为变量赋值并返回一个左值的引用，右结合。要求赋值等号左边的值是一个左值。支持连续赋值。

    ```
    a = b = c = 3; #a == b == c == 3#
    ```

- `+=`为变量加上右值的增量并返回一个左值的引用，右结合。要求赋值等号左边的值是一个左值。支持连续赋值。
- `-=`为变量右减右值并返回一个左值的引用，右结合。要求左边的值是一个左值。支持连续赋值。
- `*=`为变量算术乘右值并返回一个左值的引用，右结合。要求左边的值是一个左值。支持连续赋值。
- `/=`为变量右除右值并返回一个左值的引用，右结合。要求左边的值是一个左值。支持连续赋值。
- `%=`为变量右模右值并返回一个左值的引用，右结合。要求左边的值是一个左值。支持连续赋值。

### 2级运算符

2级运算符均是逻辑运算符，总共6个: `== >= <= > < !=`，其中：

- `==`判断左值和右值是否相等，若相等返回`1`，不相等返回`0`，左结合
- `>=`判断左值是否大于或等于右值，若为真返回`1`，为假返回`0`，左结合
- `<=`判断左值是否小于或等于右值，若为真返回`1`，为假返回`0`，左结合
- `>`判断左值是否小于右值，若为真返回`1`，为假返回`0`，左结合
- `<`判断左值是否小于右值，若为真返回`1`，为假返回`0`，左结合
- `!=`判断左值是否不等于右值，若为真返回`1`，为假返回`0`，左结合
